com.yjy.builderTest // 建造者模式
	测试煎饼果子，根据自己的喜好让老板加鸡蛋、加生菜、加里脊、加火腿、加辣椒等。
	
建造者模式概念：
	模式定义：将一个复杂对象的构建与它的表示分离，使得同样的构建过程可以创建不同的表示
	注意：与工厂模式的区别是，建造者模式更加关注与零件装配的顺序
